# React EChart 项目背景

## 1. 项目定位

React EChart 是一个面向 React 应用的 Apache ECharts 封装库。

它服务于两类场景：

1. 业务项目中的图表组件集成
2. 需要细粒度控制 ECharts 实例的高级用法

系统目标不是提供另一个图表组件库，而是在 React 生态中，以类型安全和性能优化为核心，提供对 ECharts 的完整控制能力。

---

## 2. 问题背景

当前在 React 项目中使用 ECharts 时，通常面临以下问题：

- **包体积过大**：完整引入 ECharts 会导致 bundle 体积膨胀，影响页面加载性能。
- **类型缺失**：原生 ECharts 的 TypeScript 类型支持不够完善，开发体验不佳。
- **生命周期管理复杂**：React 组件卸载时需要手动 dispose ECharts 实例，容易出现内存泄漏。
- **响应式处理繁琐**：监听容器大小变化并调用 `resize()` 需要额外的代码。
- **扩展加载困难**：按需加载 ECharts 组件（如 Grid、Tooltip）需要手动管理依赖关系。

因此，直接使用原生 ECharts API 在 React 项目中会导致代码冗余、维护困难和性能问题。

---

## 3. 核心目标

React EChart 希望建立一套可复用的图表组件能力，使集成过程具备以下特征：

- **类型安全**：每个图表组件都有完整的 TypeScript 类型定义，包括 options 和事件。
- **按需加载**：根据 options 内容自动识别并加载所需的 ECharts 扩展组件。
- **生命周期自动化**：自动管理 ECharts 实例的初始化、更新和销毁。
- **响应式支持**：内置 ResizeObserver，自动处理容器大小变化。
- **事件透传**：支持 React 风格的事件绑定（如 `onClick`），自动转换为 ECharts 事件。
- **实例可访问**：通过 ref 暴露底层 ECharts 实例，支持高级用法。

---

## 4. 核心原则

### 4.1 按需加载是默认行为

扩展组件的加载不是可选的优化，而是默认的架构设计：

- Grid、Tooltip、Legend 等组件根据 options 内容自动加载
- 不需要用户手动管理 `echarts.use()` 调用
- 未使用的扩展不会出现在最终 bundle 中

### 4.2 类型定义与运行时行为一致

TypeScript 类型不是事后补充的装饰，而是与运行时行为严格对应：

- 每个图表组件的 options 类型由 ECharts 官方类型派生
- 事件类型通过模板字符串类型自动生成
- 类型错误在编译时而非运行时被发现

### 4.3 组件封装不丢失控制能力

封装层不阻断对底层 ECharts 的访问：

- 通过 `ref.instance()` 获取原生 ECharts 实例
- 支持所有 ECharts 原生配置项
- 事件处理保持与原生 API 一致的参数结构

### 4.4 性能优化内置化

性能优化不是用户的责任，而是库的默认行为：

- 自定义 `memo` 比较函数，避免不必要的重渲染
- Resize 回调内置防抖支持
- 扩展组件异步加载，不阻塞初始渲染

---

## 5. 目标边界

### 本项目要做

- 提供常用图表类型的 React 组件封装（Bar、Line、Pie 等）。
- 提供自动扩展加载机制，优化 bundle 体积。
- 提供完整的 TypeScript 类型定义。
- 提供响应式容器大小监听。
- 提供 React 风格的事件绑定。
- 提供实例访问能力，支持高级定制。

### 本项目暂不做

- 提供图表配置的可视化编辑器。
- 提供图表数据的自动转换或聚合。
- 封装所有 ECharts 图表类型（优先覆盖常用类型）。
- 提供服务端渲染（SSR）支持。
- 提供图表动画的高级控制。
- 替代用户对 ECharts 配置的理解。

---

## 6. 成功标准

项目成功不以"封装了多少 API"衡量，而以以下能力衡量：

- 用户可以通过一行 JSX 代码渲染一个完整的图表。
- TypeScript 用户获得完整的类型提示和错误检查。
- 生产环境的 bundle 体积相比完整引入 ECharts 显著减小。
- 组件卸载后无内存泄漏。
- 容器大小变化时图表自动适应。
- 相同配置下，渲染性能与原生 ECharts 一致。

---

## 7. 一句话定义

> React EChart 是一个以按需加载优化包体积、以 TypeScript 类型保障开发体验的 React ECharts 封装库。
